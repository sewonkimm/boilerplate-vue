# 🍎개발환경구성

1. npm 사용을 위한 Node.js 설치 or yarn 설치

    yarn은 보안적인 성능이나 빌드 성능 등을 개선한 패키지 매니저

    설치 후 아래 명령어로 설치가 잘 되었는지 확인

    ```
    $ npm -v
    $ yarn -v
    ```

2. vue cli 설치

    ```
    $ yarn global add @vue/cli 
    ```

3. vue create [project-name] 으로 프로젝트 생성
